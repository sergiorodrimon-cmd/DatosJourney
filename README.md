# 📊 Automatización de Reporte de Ventas

## 🧩 Descripción del Proyecto
Este proyecto automatiza la generación de un reporte mensual de ventas a partir de datos crudos.
El objetivo es reducir el tiempo manual de análisis y entregar métricas clave listas para la toma
de decisiones.

## 🎯 Objetivo de Negocio
Brindar una visión clara del rendimiento comercial mediante KPIs actualizados, confiables
y fáciles de interpretar.

## 🗂️ Datos
Los datos provienen de archivos CSV de ventas y contienen información como:
- Fecha de venta
- Producto
- Región
- Cantidad
- Precio

## ⚙️ Proceso
1. Limpieza y validación de datos
2. Transformación de variables
3. Cálculo de KPIs
4. Generación de archivos y gráficos finales

## 📈 KPIs Calculados
- Ventas totales
- Crecimiento mensual
- Ticket promedio
- Top productos y regiones

## 🛠️ Herramientas Utilizadas
- Python
- pandas
- numpy
- matplotlib / seaborn
- Excel / CSV
